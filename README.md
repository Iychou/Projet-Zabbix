Infrastructure Monitoring with Zabbix

This project demonstrates the deployment of a professional infrastructure monitoring solution using Zabbix.
The goal is to supervise Linux servers, monitor system resources (CPU, RAM, disk), track critical services, and generate alerts in case of failures.

The project follows real enterprise practices, including a dedicated monitoring server, agents on client machines, firewall configuration, logging, and dashboards.

Technologies: Zabbix, Ubuntu Server, MariaDB, Apache, Bash

Skills demonstrated:

Linux system administration

Infrastructure monitoring

Incident prevention

Production-ready configuration
